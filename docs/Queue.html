    References to first and last elements of queue. Strictly speaking we don't need the last element (it can be inferred by traversing node.next) but having it improves performance of push.

    Add value to queue.

      public void push(T value){
    Node <T> node = new Node(value);
    if(last != null) last.next = node;
    last = node;
    if(first == null) first = node;
  }

    Add multiple values to queue.

      public void push(T... values){for(T value : values) push(value);}

    Take value from queue.

      public T pop(){
    if(first == null) return null;
    T value = first.value;
    first = first.next;
    if(first == null) last = null;
    return value;
  }
